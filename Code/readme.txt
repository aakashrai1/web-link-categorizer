Web categorizer

================================
Requirements
================================

pandas
matplotlib
sklearn
wordcloud
seaborn

================================
Steps to reproduce the results
================================

Run model.ipynb python notebook file (It already has results saved)

OR

Run following command in the project directory

python model.py


================================
Analysis
================================


The analysis.py file has the analysis code for selecting number of features, n-grams, tree depth, 
PCA and SVD, generating latent semantic analysis components, and various feature selection methods such
as chi2, mutual info, variancethresholding, forward elimination.
